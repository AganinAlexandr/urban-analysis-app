# Пример файла .env для настройки API ключей
# Скопируйте этот файл в .env и заполните ваши API ключи

# YandexGPT API ключ (опционально для LLM анализа)

# GigaChat API ключ (опционально для LLM анализа)

# Qwen API ключ (опционально для LLM анализа)
