Извлечение и анализ данных из файлов по программе городская среда
А) Необходимо извлечь из json файлов следующую общую информацию:
1. Группу объекта (группы могут быть либо в названии поля из json, либо в названии поддиректории с имеющимися файлами json)
2. Название объекта (обычно поле в json называется «name», иногда есть второе поле названия – называется «founded.name»)
3. Адрес объекта
4. Рейтинг для объекта (если есть, в json могут быть два рейтинга с названиями: «rating» и «average_rating»)
5. Счетчик запросов по api (если есть, название может быть «review_count_from_api»)
6. review_count_fetched (если есть)
7. Текст отзыва (в колонке текста отзывов сам текст отзыва может быть пустым)
8. Рейтинг отзыва (если рейтинг объекта в json встречается один раз для объекта, то далее в json для каждого отзыва в списке есть свой рейтинг отзыва. Для отзывов рейтинги представляют собой целые числа в диапазоне 1-5. Иногда, при наличии, название поля в json имеет название «star» или «stars»)
9. Текст ответа (если есть, при наличии название поля в json может называться «answer». В колонке текста ответов сам текст ответа может быть пустым)
10. Имя разместившего отзыв (поле может быть пустым, иногда поле может иметь название «user_name» или «name»)
11. Дата отзыва (Поле может называться «date»)
12. Название каталога с файлами json (у нас 3 каталога: «2gis», «yandex», «другие»)

Значения элементов для Группы объекта:
*  resident_complexes
*  schools
*  hospital_yandex
*  pharmacy_parse
*  kindergarden_parse
*  polyclinic_parse
*  schools_parse
*  university_parse
* shopmalls_parse
* hospital_2gis



Б) Нужно попытаться извлечь из текста (в случае наличия заполненного текстового поля):
1. Ключевые слова текста (1-4 слова)
2. Определить окраску сообщения (благодарность, восхищение, недовольство, предложение, может быть что-то иное)
3. выполнить преобразование текста отзыва в embedding (рекомендуется использовать вариант word2vec для работы с русскими текстами)
4. выполнить преобразование текста ответа в embedding (рекомендуется использовать вариант word2vec для работы с русскими текстами)
5. сравнить embeddings (полученных как из текста отзыва, так и текста ответов) с разметкой - рейтингами отзывов
В) Из адресов объектов установить (в предположении, что адреса находятся в Москве):
1. Административный округ города
2. Район города

Г) Визуализация данных. Предоставить следующие диаграммы:
1. Гистограмму распределения количества объектов от группы объекта (горизонтальная ось) в зависимости от категорий (каталогов с файлами: «2gis», «yandex», «другие»)
2. Гистограмму распределения количества отзывов от группы объекта (горизонтальная ось) в зависимости от категорий категорий (каталогов с файлами: «2gis», «yandex», «другие»)
3. Гистограмму распределения количества объектов от группы объекта (горизонтальная ось) в зависимости от категорий (Административных округов)
4. Гистограмму распределения количества отзывов от группы объекта (горизонтальная ось) в зависимости от категорий (Административных округов)
5. Гистограмму заполненности данными (значения, отличные от нуля или пустого текста): процент заполнения в зависимости от вида данных (рейтинг отзыва, текст отзыва, текст ответа) по горизонтальной оси в зависимости от категорий (каталогов с файлами: «2gis», «yandex», «другие»)
6. Гистограмму распределения Рейтинга отзывов от группы объекта (горизонтальная ось) в зависимости от категорий (каталогов с файлами: «2gis», «yandex», «другие»)
7. Вывести таблицу по Рейтингам отзывов, усредняя их по подгруппам:  в строках таблицы делим данные по Округам города и далее Округи по районам, столбцы таблицы содержат названия Групп объекта. Форматируем цветом от красного для «1» до зеленого для «5».
8. Представить диаграмму корреляции Рейтинга отзыва и embeddings соответствующих текстов отзыва и ответа (при их наличии).
9. Предоставить диаграмму зависимости Окраски сообщения (по итогу обработки текста отзыва) в зависимости от Группы объекта.
10. Вывести информативную диаграмму ключевых слов текстов отзывов в зависимости от Группы объекта в зависимости от категорий (каталогов с файлами: «2gis», «yandex», «другие»)
